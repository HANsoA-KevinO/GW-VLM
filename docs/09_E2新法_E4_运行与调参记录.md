## 0. 这轮在做什么

先用新设计(统一基础模型 + 高斯后验头)**只在 90 真实事件上**重练参数估计(= **new-E2**),作为 E4(加注入数据)的**对照组**,以隔离三件事:
- E2-old(bin 分类、纯图,已有) → new-E2(后验头 + 应变):**形态 + 应变**的贡献
- new-E2 → E4(真实 + 注入):**注入数据**的贡献

后验头对每个参数输出 μ(均值)+ σ(不确定度),NLL 训练。**要盯的失败模式 = 方差塌缩**(σ 被压到极小、假装自信 → NLL 爆炸 → 既不可信也可能干扰检测)。这是 E2-bins"众数塌缩"的回归版亲戚。

## 6. 方差塌缩监控(进行中)

- 训练步打印是**每窗口最后一个样本**(非步均值)→ 噪声大;`nll=0` = 该样本是负样本(被 mask)。
- 早期正样本 nll:4.71 / 5.84 / 3.84 / 4.45 / **18.93(step30 尖刺)** / 3.55 → 一个尖刺 + 其余 3.5-6,**疑似噪声 + 一个难样本**,未确诊塌缩。
- clamp 现为 [-7,5](偏松,允许 σ→0.0009)。
- **决策点(step120 窗口诊断)**:比较早窗(≤40)vs 晚窗(80-120)正样本 nll 均值 + 爆炸(>15)计数。
  - 晚 < 早 且 爆炸少 → **放行**跑完。
  - 晚 ≥ 早 或 爆炸多 → **停,重启**改 **logσ_min -7→-2** + **head_lr 1e-3→5e-4**(标准防塌缩;同样应用到 E4,对照不受影响)。

## 7. step120 判定(19:17)→ 放行,不调参

> 说明:这是**训练** nll(且是单样本噪声读数),只能说明"在学、没塌"。真正回答"是否学到可校准的后验、还是过拟合 90 事件"的是**测试集评估**(NLL/MAE/bin等价/PIT 覆盖率),见完成后第 8 节。

## 8. new-E2 评估结果(2026-06-30 01:12,测试集全真实 n=135 正样本)

**检测 ROC-AUC = 0.9477**(PR-AUC 0.9555,R@0.5=0.807,R@FPR5%=0.793)→ 与 E1(0.94)/E2-old(0.959)持平,**后验头+应变没拖累检测**。

**参数**(bin等价 = 把回归点估计再分 bin,对照 E2-old 直接优化的 bin 精确;NLL/MAE/覆盖率是后验头原生口径):

| 参数 | E2-old bin精确 | new-E2 bin等价 | 随机 | NLL | MAE | cov50/cov90 |
|---|---|---|---|---|---|---|
| chirp_mass | 0.239 | **0.400 ⬆** | 0.143 | 1.466 | 10.9 M☉ | 0.73 / 0.93 |
| distance | 0.442 | **0.200 ⬇** | 0.167 | 1.449 | 1360 Mpc | 0.27 / 0.93 |
| chi_eff | 0.265 | 0.267 ≈ | 0.2 | 1.329 | 0.107 | 0.27 / 1.00 |

**解读(回答"形态 vs 数据墙"):**
- **chirp_mass = 形态问题,被新法修好**:0.24→0.40(远超随机),校准良好(cov 0.73/0.93)。质量在 chirp 形态里信息强,连续后验头+应变读得比粗 bin 好。← 新法的胜利。
- **distance = 退步**:0.44→0.20(勉强超随机),MAE 1360 Mpc 差,cov50=0.27 过自信(σ 偏窄、μ 向均值缩)。distance 退化(distance-inclination 简并),回归头易"回归均值";E2-old 的 0.44 部分占了"直接优化 bin + 与 SNR 相关"的便宜。但 MAE 客观地差。
- **chi_eff = 数据墙**:≈E2-old≈随机,MAE 0.107≈std(基本预测均值)。自旋最弱最难,90 事件喂不动。预期之中。
- **校准**:90% 区间基本对(cov90 0.93-1.0);50% 区间对 distance/chi_eff 过窄(过自信)——难参数上小数据过自信。

**→ 对 E4 的指向**:形态修复对强信号参数(chirp_mass)已奏效;distance/chi_eff 卡在数据墙——**E4(注入加数据)正是要验证"加数据能否把 distance/chi_eff 抬离墙"**。这强化了 E4 跑满 EP=2 的价值(那才是核心问题)。

产物:`output/runs/e2_unified_real/unified_eval.json`。

## 10. 参数重设计(detector-frame)+ new-E2 v2 结果 + 均匀采样转向

### 10a. 参考系修复 + 目标重选(提交 c061794)
用户审查暴露两个 bug,E4 前必修:
1. **参考系**:events.csv 的 detector-frame `chirp_mass` 列 100% 空,加载用了 source-frame(= detector ÷(1+z),把红移除掉);而波形/Q图编码 detector-frame。更糟:注入是 detector-frame → 真实(source)与注入(detector)标签不同系(中位差30%、最高2.18×),E4 混训=矛盾标签。
2. **目标**:distance 单探测器与 inclination 简并、信息欠定(用户拍板剔除);改目标为 detector-frame **{chirp_mass, total_mass, chi_eff}**。
修复:真实质量→detector(source×(1+z));注入加 total_mass;标签清洗(低SNR/离群事件 param_eval=False,只影响参数评分,不动检测/训练)。重建 e5 train 9674、e5_real 2394,抽验 det=source×(1+z) 精确、真实与注入同系。

### 10b. new-E2 v2 结果(detector-frame,EP=2,可靠测试 n=117)—— 关键负结果
| | bin-exact | 随机 | MAE | cov50/cov90 |
|---|---|---|---|---|
| 检测 ROC-AUC | **0.922** | — | — | R@0.5=0.75 |
| chirp_mass | **0.154** | 0.143 | 19.4 | 0.54/0.92 |
| total_mass | 0.385 | 0.167 | 43.3 | 0.54/0.92 |
| chi_eff | 0.308 | 0.2 | 0.095 | 0.31/1.00 |
| 派生 m1/m2/q MAE | 24.7/20.0/0.18 | | | |

**核心发现:v1 的 "chirp 0.40" 是假象。** 修对参考系后 chirp 掉到 0.154(≈随机):
- v1 source-frame 范围集中(中位24)→ bin-exact 0.40 其实是**猜众数 bin**;v2 detector-frame 范围散(中位32、上到131)→ 猜众数命中率掉到≈随机。
- 各参数 bin-exact ≈ 各自众数 bin 占比 + cov 接近名义 → **模型在预测先验/均值,没从形态读出质量**。
- **经典基线 val_nll 爆炸(5.6→1325,best@ep0=未训)**:90 真实事件上瞬间过拟合,其"最佳"=均值预测;VLM 数字与之几乎相同 → 两者都≈均值预测。
- **结论:数据墙坐实**——90 真实事件上基础模型检测强(0.92)、但参数连 chirp 都读不出(众数预测)。v1 的乐观被参考系修正证伪。
