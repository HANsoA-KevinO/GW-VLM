## 0. 这轮在做什么

先用新设计(统一基础模型 + 高斯后验头)**只在 90 真实事件上**重练参数估计(= **new-E2**),作为 E4(加注入数据)的**对照组**,以隔离三件事:
- E2-old(bin 分类、纯图,已有) → new-E2(后验头 + 应变):**形态 + 应变**的贡献
- new-E2 → E4(真实 + 注入):**注入数据**的贡献

后验头对每个参数输出 μ(均值)+ σ(不确定度),NLL 训练。**要盯的失败模式 = 方差塌缩**(σ 被压到极小、假装自信 → NLL 爆炸 → 既不可信也可能干扰检测)。这是 E2-bins"众数塌缩"的回归版亲戚。

## 6. 方差塌缩监控(进行中)

- 训练步打印是**每窗口最后一个样本**(非步均值)→ 噪声大;`nll=0` = 该样本是负样本(被 mask)。
- 早期正样本 nll:4.71 / 5.84 / 3.84 / 4.45 / **18.93(step30 尖刺)** / 3.55 → 一个尖刺 + 其余 3.5-6,**疑似噪声 + 一个难样本**,未确诊塌缩。
- clamp 现为 [-7,5](偏松,允许 σ→0.0009)。
- **决策点(step120 窗口诊断)**:比较早窗(≤40)vs 晚窗(80-120)正样本 nll 均值 + 爆炸(>15)计数。
  - 晚 < 早 且 爆炸少 → **放行**跑完。
  - 晚 ≥ 早 或 爆炸多 → **停,重启**改 **logσ_min -7→-2** + **head_lr 1e-3→5e-4**(标准防塌缩;同样应用到 E4,对照不受影响)。
