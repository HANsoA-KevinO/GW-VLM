# 训练方法前沿调研与学习笔记(2024–2026)

> 目的:在 E1 检测见顶(ROC-AUC 0.92–0.94)后,系统调研"训练方法"层面还有哪些可试的改进,并为非训练专家梳理出可学习、可决策的清单。
> 方法:5 路并行文献调研(先进 PEFT / 难样本与损失与课程 / 数据中心 / VLM 专属 / GW 检测 ML 领域),均带一手来源与证据强度标注。
> 日期:2026-06。

---

## 0. 一页纸结论(先看这个)

**最重要的一句话**:多条独立证据指向——**我们 0.94 的天花板,主要是"任务/信息上限",不是"LoRA 配置没调好"**。引力波"单探测器频谱图检测"在公开文献里大致就在 AUC 0.85–0.92 见顶,我们已经在/略超这个区间。所以:

- **换 LoRA 变体(DoRA/PiSSA/…)基本无用**——公平调学习率后它们的"涨点"几乎消失(这是 2026 一篇专门复现论文的结论)。别在这上面花时间。
- **真正撬动天花板的是"信息"而非"训练技巧"**:① 注入数据 + 按 SNR 课程(造我们最缺的低 SNR 正样本);② H1+L1 双探测器一起喂;③ 找回相位信息(频谱图把相位丢了,而匹配滤波/1D 方法靠相位);④ 用判别式 logit 头取代生成式 YES/NO。
- **但确实有一批"纯训练方法"的便宜改进值得先做**(下面 A 档):决策 token 损失聚焦、概率抽取审计、权重平均、SNR 课程、LoRA 卫生。它们便宜、低风险,可能各拿几个点 + 让评估更可信。

**一句忠告**(多个调研共同强调):先跑一个**调好的 CNN/ViT 或 1D 基线**,确认 VLM 看图这条路相对传统方法到底有没有竞争力——GW glitch 分类 CNN 常 >98.8%,而对短促 BBH chirp,1D strain 通常比频谱图更敏感。VLM 路线更现实的卖点是**可解释 + 多模态/集成**,而不是"在最弱信号上击败 1D"。

## 1. 心智模型:"训练方法"到底有哪些可动的旋钮

把"训练方法"拆成 5 层,改进机会主要在 (b)(a)(c),不在 (d):

| 层 | 是什么 | 我们的现状 | 改进空间 |
|---|---|---|---|
| (a) **损失**:在什么上、怎么算 loss | 整串 JSON 的 token 级交叉熵 | 模板 token 稀释了 YES/NO 决策 token | **大**(损失聚焦/加权/balanced softmax) |
| (b) **数据**:喂什么、什么顺序 | 2394 真实事件+jitter,随机顺序 | 缺低 SNR 正样本;无课程 | **最大**(注入+SNR课程+双探测器) |
| (c) **输出/读取**:怎么把生成变成分数 | 取决策位 YES/NO 的 logit → P(YES) | 可能没合并大小写/空格变体 | **中**(抽取审计、软平均、校准) |
| (d) **PEFT 参数化**:怎么微调 | bf16 LoRA r32 | 已接近最优 | **小**(变体基本无用;查 target/LR/早停) |
| (e) **测试时**:推理时技巧 | 单次贪心 | 无 TTA/集成 | **中**(物理合法 TTA、权重平均) |

**三个反复出现的关键认知:**
1. **ROC-AUC 只看"排序"。** 温度缩放/Platt 这类单调校准**不改 AUC**,只让概率可信、便于卡阈值。要涨 AUC,必须**改善排序**(更好的数据/特征)或**修正读分数的方式**,而不是事后校准。
2. **"难样本"和"标错的样本"在损失上长得一样。** SNR<8 接近噪声底,人标本身不可靠,所以**无脑专攻最难样本(裸 OHEM / 高 γ focal / SMOTE)会把模型往错标样本上拽**。要用"噪声感知"的做法。
3. **弱信号训练能泛化到强信号,反之不行**(Schäfer 2021)。所以课程要**早期就暴露低 SNR**,而不是"先强后弱"。

## 2. A 档:便宜的"纯训练方法"改进(建议先做,低风险)

> 这些直接回应你"在训练方法上改进"的诉求。成本低、风险小,大多能叠加。

### A1. 决策 token 损失聚焦 ⭐(最对症,几乎零成本)
- **是什么**:训练时把固定 JSON 模板 token(`{"detection": "`、`"}`)用 `ignore_index` mask 掉,**只在 YES/NO 值 token 上算损失**(或给它显著更高权重)。
- **为什么**:正是我们上次讨论的"决策信号被一堆模板 token 稀释"问题——它有名字叫 **loss dilution**,且有论文证实:**答案越短、提示越长(正是我们),"怎么处理模板 token 的损失"越要紧**(EMNLP 2024 "Does Prompt Loss Matter?";SFTKey 2025)。
- **成本**:极低(label 张量上加一个 mask)。**证据**:中–强。
- **怎么做**:在 collator 里把 assistant 答案中模板部分的 label 设为 -100,只保留 YES/NO(及紧邻字段名)。

### A2. 概率抽取审计 + 软平均 + 校准 ⭐(可能在静默漏分)
- **是什么**:三件事——① 取分数时把 `YES/Yes/ YES/yes` 等大小写+前导空格变体的概率**求和**再做 2 类归一化 `P(YES)/(P(YES)+P(NO))`;② 确认读的是 JSON 里 answer 值的那个决策 token,不是标点;③ 多次/多视图时**平均 P(YES)**(软投票),绝不要对 YES/NO 硬投票(硬投票把连续分数塌成离散计数、毁 AUC)。
- **为什么**:token 变体分散概率质量会**改变排序→可能直接影响 AUC**(不像事后校准)。这是潜在的免费分数。
- **成本**:极低。**证据**:中(机制)+ 强(已知工程坑)。
- **附**:温度缩放对 VLM 一致有效、跨分布漂移仍有效——但记住它**不涨 AUC**,只让阈值可信(尤其将来部署/换分布时)。

### A3. LoRA 卫生检查(做对就行,免费)
- **target_modules 必须挂到所有线性层(尤其 MLP:gate/up/down),不能只挂 attention 的 q,v。** 这是 PEFT 里证据最强、最一致的一条。→ **立刻检查我们的配置。**
- LoRA 学习率约是全参微调的 **~10×**(新手常踩坑);rank 16–32 起步(我们 r32 OK);小数据用**早停**(按验证集,patience 2–3,存最佳)——我们"多 epoch 无增益"正是过拟合信号。
- **小数据下 LoRA 比全参更稳、更不易崩**,所以别为了涨点改全参。
- **证据**:强(Thinking Machines "LoRA Without Regret";LoRA Learns Less and Forgets Less, TMLR 2024)。

### A4. 权重平均 / Model Soup(免费拿稳健性+一点点 AUC)
- **是什么**:把我们已经训出来的几个 0.90–0.94 的模型**权重直接平均**(model soup),或把微调权重与原始预训练权重插值(WiSE-FT)。推理零额外成本。
- **为什么**:多个超参微调出的模型平均,精度和 OOD 鲁棒性都涨,且不增推理成本(Model Soups, 2022;WiSE-FT)。
- **成本**:低。**证据**:强。注意:不同架构(Qwen vs Gemma)不能直接平均,只能同架构/同基座的多个微调之间平均。

### A5. 按 SNR 的课程学习(便宜,但务必和随机顺序对比)
- **是什么**:给训练样本按 SNR 排序,**早期就混入/侧重低 SNR**(而非"先强后弱"),或让 SNR 下限随训练逐步下移。
- **为什么**:GW 领域标准做法(AResGW 等);且 CurBench(ICML 2024)证明**课程学习在"噪声"场景下最可能有用**——低 SNR≈高噪声,正中。但要注意 LLM 课程普遍增益小、不一定泛化,**必须和随机顺序做对照 ablation**。
- **关键纠偏**:Schäfer 2021——**弱信号训练能泛化到强,反之不行**,所以重点是"早暴露弱",而非经典的"先易后难"。也可试 self-paced(用模型自身 loss 推难度,不需 SNR 标签)。
- **成本**:低(改采样顺序)。**证据**:中(领域有先例但多未单独 ablate)。

## 3. B 档:真正的大杠杆(更费工,但天花板更高)

> 这些严格说超出"纯训练方法",偏数据/输入/架构——但**它们才是突破 0.94 的真正希望**。

### B1. 注入数据 + SNR 课程 ⭐⭐(领域公认的弱信号正解)
- **是什么**:用模拟波形(PyCBC/LALSuite)注入**真实 H1/L1 噪声**,可控设定 SNR,无限量造"标签 100% 可信"的低 SNR 正样本;配 SNR 课程。这就是我们一直说的 E3/E4。
- **为什么**:唯一能**直接、无限量**制造我们最缺的低 SNR 正样本的手段,是 GW ML 的标准配方(MLGWSC-1、AResGW)。采样按 **chirp-distance 均匀**(而非 SNR 均匀)更利于弱信号。
- **关键坑**:合成谱必须与真实谱**同分布**(同 Q-transform 参数、viridis、分辨率),否则模型学的是"合成 vs 真实"的捷径;验证集必须用真实事件。
- **成本**:中(建流水线,主要吃 CPU)。**证据**:强(领域 SOTA 标准)。

### B2. 双探测器 H1+L1 联合输入 ⭐⭐
- **是什么**:把 H1、L1 一起喂(双通道/双图 + 时延信息),而非单探测器单图。
- **为什么**:**弱信号最大的物理杠杆**——双探测器符合在整个 SNR 范围系统性抬高检测曲线,并压制单探测器 glitch 假信号(glitch 在频谱图上常像信号)。我们当前是"每(事件,探测器)独立成样本",属单探测器——这是结构性改进点。
- **成本**:中(改数据组织 + 输入格式)。**证据**:强。
- **注**:这与我们 docs/02 的"方案 A 单探测器、不堆叠"决策冲突,需要重新权衡(当初为部署灵活性选了单探测器)。

### B3. 找回相位 / 原始 whitened strain(或与 1D 混合)
- **是什么**:频谱图丢了相位信息,而匹配滤波/1D 深网正是靠相位相干。可考虑:输入补充原始 whitened strain,或做 VLM(频谱图分支)+ 1D-CNN(strain 分支)集成。
- **为什么**:1D+2D 集成在文献里全面胜出(arXiv:2312.04855);对短促 BBH chirp,1D 通常更敏感。
- **成本**:中–高。**证据**:强。

### B4. 判别式 logit 头取代生成式 YES/NO
- **是什么**:在 VLM 的视觉/隐藏特征上接一个二分类 logit 头(线性探针),而不是让模型"生成"YES/NO token。
- **为什么**:生成式输出把分数压成近二值,对需要连续分数的 ROC-AUC 不利;判别头几乎总给更好的连续分数和更高 AUC(生成式 vs 判别式分类多项研究一致)。
- **成本**:中(改输出头 + 训练目标)。**证据**:强(但非 GW 直接实验)。
- **权衡**:会牺牲 VLM 的"可解释/多任务"卖点,变成纯分类器——要想清楚项目定位。

## 4. C 档:重新定位 + 必要的基线

- **跑一个调好的 CNN/ViT(2D 频谱图)和/或 1D-strain 基线**:用来确认"信息上限"和"VLM 路线是否真有优势"。如果一个朴素 CNN 就 0.95+,说明瓶颈是 VLM 表达而非信息;如果 CNN 也卡在 0.92,说明是信息墙。**这是判断后续往哪投的关键一步。**
- **评估口径向 GW 社区对齐**:别只报 AUC(它把无关的高 FPR 区也等权积分);重点报**固定低 FAR(如 1/月)下的 recall** + **按 SNR 分箱的 detection efficiency 曲线**。我们"低 FPR recall + SNR 诊断"方向已对,但可更标准化(必要时用 PyCBC `volume_montecarlo` 报灵敏距离)。

## 5. 避坑清单(有证据表明对我们有害或无用)

- **❌ Label smoothing**:会损害我们最依赖的"按置信度排序卡阈值/识别弱信号"的能力(ICLR 2025)。
- **❌ 分类前加长 CoT 自由推理**:感知类任务上常掉点 + 增幻觉(GenCLS++;To CoT or not to CoT)。若要"先描述特征再判",用**结构化短描述(CCoT 式,纯 prompt)**,别长篇。
- **❌ 裸 OHEM / 高 γ focal / 普通 SMOTE / 无脑难负挖掘**:在 SNR<8 标注不可靠区会放大错标。要用就用噪声感知版(GHM、positive-aware mining、meta-reweighting)。
- **❌ 几何/时频-mask 类 TTA**(翻转/旋转/裁剪/频率掩码):破坏频谱图物理,医学 TTA 实测 12 个里 11 个变差。只用物理合法视图(小时移、温和多尺度、亮度/对比抖动)且永远保留原图、每次验 AUC。
- **❌ 把校准当成能提分离度**:温度/Platt 不改 AUC。
- **❌ LoRA 变体(DoRA/PiSSA/AdaLoRA/VeRA/MoRA/OLoRA/NEFTune)**:公平调参后无增益或不适用;最多顺手试 DoRA(对 rank 鲁棒)、LoRA+(零成本),别抱期望。
- **❌ RLVR/GRPO 提准确率、RLHF/DPO**:RL "锐化而非拓展"能力,在 Qwen 上还有"虚假奖励"假象;我们瓶颈是能力/信息缺口,正是 RL 最帮不上的。唯一例外是少样本场景的 Visual-RFT,但频谱图 OOD、医学变体收敛难。

## 7. 证据强弱与诚实提示

- **强且基础牢**:LoRA 变体祛魅(LR 决定一切)、target 挂 MLP、Model Soups、Focal/logit-adjustment 基础、单调校准不改 AUC、MLGWSC-1 的 95%/70%、AResGW 超匹配滤波、Schäfer 2021"弱→强泛化"、注入+SNR课程为领域标准、"Does Prompt Loss Matter?"的 short-completion 结论。
- **中等/需自己 ablate**:决策token聚焦在我们任务上的确切增益、SNR课程的独立贡献(领域先例多未单独 ablate)、判别头在 GW 上的增益、双探测器对 VLM 路线的具体增益。
- **空白**:**没有任何文献是针对"引力波 VLM 生成式检测"的**;上述机制迁移合理,但在我们任务上的确切收益**未验证,需自己做对照实验**。
